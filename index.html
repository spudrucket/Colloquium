Colloquium

An open source qualitative ethnographic interview coding tool.

View on GitHub Download .zip Download .tar.gz

Colloquium QDA is a tool for custom coding and analyzing qualitative ethnographic interviews. To run, make sure you first have Java 8 or later installed.

Current features: Add informants Add interviews and translations Create custom codes/tags Code interviews Search using basic Boolean operators Export, Backup and Restore database

Colloquium QDA is an open source cross-platform Java Swing app utilizing an embedded Java DB with Lucene integrated search.
